🏠 Simple Rent Calculator

A basic Python program to calculate how much each person needs to pay when sharing rent, food, and electricity expenses.

📌 Features
Calculates total electricity bill
Splits total expenses among roommates
Handles invalid inputs
Provides accurate results with decimal values
🧮 Formula Used

Total Expense = Rent + Food + (Electricity Units × Charge per Unit)

Per Person Share = Total Expense ÷ Number of Person

📥 Inputs Required
Flat rent amount
Food expenses
Electricity units consumed
Charge per unit
Number of persons
📤 Output
Total expense
Amount each person needs to pay
